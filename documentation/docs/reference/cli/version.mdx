# amctl version

Print the `amctl` version, commit, and build date.

```
amctl version
```

## Example

```bash
amctl version
```

Sample output:

```
amctl 0.1.0 (commit abc1234, built 2026-05-15T05:30:00Z)
```

## Options inherited from parent commands

| Name | Description |
|------|-------------|
| `--json` | Output the version as a JSON envelope. |
